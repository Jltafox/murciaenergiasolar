# Ideas de Diseño — Murcia Energía Solar

## Contexto
Landing page de alta conversión para empresa instaladora de placas solares en Murcia. Mobile-first, CRO optimizado, arquitectura de silos SEO. El objetivo principal es generar leads (solicitud de presupuesto gratuito).

## Opción A — "Sol Mediterráneo Industrial"

**Design Movement:** Brutalismo suavizado + Mediterráneo moderno

**Core Principles:**
1. Contraste brutal entre bloques de color sólido (negro carbón, amarillo solar intenso, blanco roto) para crear impacto visual inmediato.
2. Tipografía pesada y asimétrica que transmite solidez y confianza técnica.
3. Texturas de hormigón y acero en fondos para evocar la robustez de las instalaciones.
4. Datos numéricos en tipografía gigante como elementos decorativos y de prueba social.

**Color Philosophy:** Negro carbón (#1A1A1A) como base dominante, amarillo solar intenso (#F5C518) como acento de conversión, blanco roto (#F7F4EF) para zonas de respiro. La paleta evoca paneles solares reales: negro del panel, amarillo del sol, blanco del cielo murciano.

**Layout Paradigm:** Bloques asimétricos con cortes diagonales. El hero ocupa el 100vh con texto a la izquierda y un panel solar en perspectiva a la derecha. Las secciones se alternan entre fondo negro y blanco con bordes diagonales.

**Signature Elements:**
- Números gigantes (ej. "3.300 horas de sol") como elementos visuales de fondo.
- Bordes diagonales entre secciones (clip-path).
- Iconografía técnica de línea gruesa.

**Interaction Philosophy:** Hover states con desplazamiento de color (fill animation). El CTA principal tiene un efecto de "carga solar" al hacer hover.

**Animation:** Entrada de secciones con slide-up + fade. El contador de ahorro anima desde 0. El hero tiene un parallax sutil en el panel solar.

**Typography System:** `Bebas Neue` para headlines (display, impacto máximo) + `DM Sans` para body (legibilidad, modernidad). Jerarquía: 72px/48px/24px/16px.

## Opción B — "Claridad Solar Mediterránea" ✅ SELECCIONADA

**Design Movement:** Modernismo Mediterráneo Limpio — inspirado en la arquitectura blanca de Murcia, el sol y el mar.

**Core Principles:**
1. Claridad radical: cada elemento tiene un propósito de conversión. Sin decoración innecesaria.
2. Calidez solar: la paleta evoca el sol de Murcia sin recurrir a clichés fotovoltaicos.
3. Confianza técnica: datos reales, certificaciones visibles, testimonios con foto.
4. Velocidad de decisión: el usuario puede solicitar presupuesto en menos de 3 scrolls.

**Color Philosophy:** Naranja solar cálido (#E8650A) como color de acción y conversión, azul cielo profundo (#1B3A5C) como color de confianza y autoridad, blanco puro (#FFFFFF) y gris arena (#F8F6F2) como fondos. El naranja evoca el sol de Murcia; el azul, el cielo mediterráneo y la solidez técnica.

**Layout Paradigm:** Layout asimétrico de dos columnas en desktop con un hero que divide la pantalla: texto e impacto a la izquierda (60%), formulario de captación a la derecha (40%). En mobile, el formulario sube al primer scroll. Las secciones usan un grid de tarjetas con sombras suaves.

**Signature Elements:**
- Barra de progreso animada "Tu ahorro estimado" que se completa al hacer scroll.
- Tabla de bonificaciones IBI por barrio de Murcia (elemento diferenciador clave del análisis).
- Contador de proyectos completados en Murcia animado.

**Interaction Philosophy:** Micro-interacciones en cada CTA. El formulario tiene validación en tiempo real con feedback visual. Las tarjetas de servicio se elevan con sombra al hover.

**Animation:** Framer Motion para entradas de sección (stagger de elementos). El número de ahorro anual se cuenta desde 0 al entrar en viewport. El hero tiene un gradiente animado sutil.

**Typography System:** `Sora` para headlines (moderno, técnico, mediterráneo) + `Inter` para body. Jerarquía clara: 56px hero / 36px sección / 20px subtítulo / 16px body.

## Opción C — "Energía Oscura Premium"

**Design Movement:** Dark Mode Premium + Energía Futurista

**Core Principles:**
1. Fondo oscuro que hace que los elementos de color (naranja, verde eléctrico) brillen con intensidad.
2. Glassmorphism en tarjetas y formularios para dar profundidad y modernidad.
3. Gradientes de energía (naranja → amarillo → verde) para evocar la transformación energética.
4. Tipografía geométrica que transmite precisión tecnológica.

**Color Philosophy:** Fondo casi negro (#0D1117), naranja energético (#FF6B35) como acento principal, verde sostenible (#00D4AA) como acento secundario, blanco puro para texto. La paleta evoca paneles solares de noche y la tecnología de punta.

**Layout Paradigm:** Full-width con secciones de pantalla completa. Hero con vídeo de fondo o imagen de paneles solares al atardecer. Cards con glassmorphism flotando sobre el fondo oscuro.

**Signature Elements:**
- Glassmorphism en el formulario de captación.
- Gradiente de energía animado en el hero.
- Iconos 3D de paneles solares generados con IA.

**Interaction Philosophy:** Efectos de brillo (glow) en CTAs al hover. Partículas de luz solar flotando en el hero.

**Animation:** Partículas de energía solar en el hero. Gradiente animado en el fondo. Entrada de elementos con efecto de "encendido".

**Typography System:** `Space Grotesk` para headlines + `IBM Plex Sans` para body. Tipografía técnica y futurista.
